# 电路计算工具五合一使用指南

## 工具内容
1. 电路相量极标式加法器
2. 电路相量直标式乘法器
3. 反三角函数计算器
4. 相量极标式转直标式
5. 相量直标式转极标式

## 使用方法
1. Clone或Fork
2. 解压缩
3. 用Code::blocks打开
4. Enjoy!

## 质量声明
1. 经过一定量的测试
2. 但一定会有bugs
3. 不喜勿喷
4. 杀鸡焉用牛刀（太简单的计算徒手算就行了）
