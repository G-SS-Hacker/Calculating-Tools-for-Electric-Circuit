## 工具内容

1. 电路相量极标式加法器
2. 电路相量直标式乘法器
3. 反三角函数计算器
4. 相量极标式转直标式
5. 相量直标式转极标式

## 使用方法

1. `Clone`或`Fork`
2. 解压缩
3. 用`Code::blocks`打开
4. Enjoy!

## 质量声明
